# SocietyEase
Named the project
**SocietyEase** is an Apartment Association Management System developed as part of our Software Engineering project. 
## Project Objective

The objective of SocietyEase is to simplify apartment association management by providing a centralized platform for residents, association members, and administrators. The system aims to reduce missed updates, improve complaint tracking, manage dues efficiently, and maintain proper records.

## Core Features

* Membership Management
* Complaint and Maintenance Management
* Invoice and Payment Tracking
* User-friendly Frontend Interface
* Neighbor Conflict Resolver - Stanalone Feature

## Team Workflow

The team follows a structured GitHub workflow:

1. Each team member works on a separate branch.
2. Changes are pushed to the respective branch.
3. Pull Requests are created for review.
4. The Project Manager reviews the Pull Requests.
5. Approved changes are merged into the main branch.
6. The main branch is maintained as the stable version.

## Project Management

The team uses a GitHub Projects Kanban Board to track tasks. The board includes the following columns:

* Backlog
* To Do
* In Progress
* Review
* Done

Each task is assigned to a responsible team member and moved across the board based on its progress.
